# The Observer

A browser game about watching, deducing, sabotaging, and socially surviving in a decaying terminal city.

## Concept

You play as someone working at a clandestine observation terminal. People enter, speak, lie, perform, hide things, and try to manipulate each other. Your role is to read the pattern behind the speech.

### Core Loop

1. **New figures enter the system** — through chat, terminal, calls, files, logs, small dialogues
2. **You observe** — each character speaks differently. Some talk too much. Others too little. Some try to seem smart. Some want to dominate the conversation.
3. **You collect signals** — not obvious "proofs", but Dexlike things:
   - tone shifts
   - overused words
   - temporal contradictions
   - false certainty
   - status worship
   - suspicious silence
   - displaced anger
   - jokes used as escape
4. **You build a mental map** — an investigative board:
   - who fears whom
   - who wants what
   - who lies badly
   - who's bluffing
   - who's useful
   - who's disposable
5. **You act** — expose, pressure, manipulate, isolate, ask the exact question, plant doubt, play people against each other, or stay quiet and watch the circus burn
6. **The system responds** — each choice alters reputation, paranoia, access to areas, and NPC behavior

### The Hook

The game doesn't treat the player as a moral hero. You don't "save the world." You just understand the environment better than it understands itself.

> The fantasy: "I'm the only one here seeing the pattern behind the smoke."

## Tech Stack

- Single HTML file with embedded CSS/JS
- No frameworks — pure vanilla
- Procedural case generation
- Local storage for persistence
